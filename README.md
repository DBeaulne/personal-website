## Installation Steps

- Clone this project 'git clone https://github.com/DBeaulne/personal-website.git'
- Go to folder 'cd personal-website'
- Run 'npm install' or 'npm i'
- Run 'npm start'
- Open your favourite broswer and navigate to 'http://localhost:3000/'
